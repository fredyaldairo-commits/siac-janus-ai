      📖 Pipeline Paso a Paso
      Knowledge

      ① Limpieza de Datos

      Se eliminan filas con valores nulos en la variable objetivo. Los valores faltantes en columnas numéricas se reemplazan por la mediana (robusta a outliers) y en categóricas por la moda.

      mediana(X) = valor central ordenado NaNnum → mediana(columna) NaNcat → moda(columna)

      ② Codificación

      Cualitativas → numéricas. Binarias se mapean a 0/1, categorías múltiples a One-Hot con drop_first para evitar multicolinealidad.

      f("Sí")=1 | f("No")=0 get_dummies(ciudad, drop_first=True)

      ③ Entrenamiento

      División 80/20 con estratificación. Datos escalados con StandardScaler. Modelo ajustado por gradiente o ensemble.

      min J(θ) = −1/N Σ[y·log(ŷ) + (1−y)·log(1−ŷ)]

      ④ Predicción

      Sobre el 20% no visto se calcula P(pago). Umbral 0.5 separa clases (ajustable).

      ŷ = P(Y=1|X) → ŷ ≥ 0.5: paga

      ⑤ Evaluación

      Accuracy, AUC-ROC, Gini, Precisión, Recall, F1.

      Accuracy = (VP+VN)/N Gini = 2·AUC − 1

      ⑥ Juan

      Cliente sintético = mediana de cada feature → vector → modelo → P(pago).

      Juan = [med(x₁), med(x₂), …] → scaler → modelo → P
      🧮 Fórmulas Matemáticas

      REGRESIÓN LOGÍSTICA

      P(Y=1) = 1 / (1 + e−(β₀ + β₁X₁ + … + βₙXₙ)) # σ(z) = sigmoide

      PROBIT

      P(Y=1) = Φ(β₀ + β₁X₁ + …) # Φ = CDF Normal

      RANDOM FOREST

      ŷ = (1/B) Σᵦ Tᵦ(x) Impureza Gini = 1 − Σpᵢ²

      XGBOOST

      F(x) = Σ fₜ(x) Obj = Σ l(yᵢ,ŷᵢ) + Ω(fₜ)

      RED NEURONAL

      a⁽ˡ⁾ = σ(W⁽ˡ⁾·a⁽ˡ⁻¹⁾ + b⁽ˡ⁾) Backprop: ∂J/∂W

      MÉTRICAS

      AUC = P(score(+) > score(−)) Gini = 2·AUC − 1 F1 = 2·P·R/(P+R)
